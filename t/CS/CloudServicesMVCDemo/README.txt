This demo utilizes the Microsoft MVC architecture in conjunction with the LEADTOOLS Cloud Services framework.

In order to run this demo you must provide the following values in the Configuration.cs file located in the Helper folder.

	1) LicenseFilePath -- The path to your LEADTOOLS license file.

	2) LicenseKey -- The contents of your LEADTOOLS license key file.  This parameter does not expect a file path, but the actual contents of the developer key file. 

	3) OCREnginePath -- The path to the directory containing the LEADTOOLS OCR Engine runtime files. By default, these files can be found
	in the following sub folder in your LEADTOOLS installation : <LEADTOOLS Folder>\Bin\Common\OcrLEADRuntime
