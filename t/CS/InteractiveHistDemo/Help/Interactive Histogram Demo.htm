The Interactive Histogram Demo

 

 

The interactive histogram demo provides powerful and comprehensive tools that allow the user to adjust any image histogram and alter, enhance or segment its information, it also contains many histogram statistical information that can be helpful to analyze the image and making further adjustment if needed.

 

There are multiple tools that are used when applying the interactive histogram.

 

To open the interactive histogram dialog, do the following:

 

1)       From File à Open.

3)       Open any Image.

4)       From analysis à Interactive Histogram (or press Ctrl + I )

 

 

The above is a screenshot of the interactive histogram dialog. The following is a detailed look at the dialog:

 

(1)     The histogram window, which is a window that have the histogram painted on it. The histogram range depends on the image itself, if the image is:

 

a.       16 bit grayscale image, the range will be from 0 à 65535

b.       12 bit image, the range will be from 0 à 4095.

c.       8 bit image, the range will be from 0 à 255.

d.       24 bit image, the range will be from 0 à 255

e.       48 or 64 bit image, the range will be from 0 à 65535

 

The histogram window contains two poles that can be grabbed using the left mouse button and moved independently, those poles identify the histogram selection range, which will affect other tools and statistical information

 

Also, the histogram window can be adjusted to zoom in and out in order to allow clear view of the histogram (this is useful when handling the 16 or 12 bit images). This can be done by doing a right click on the histogram window, three options are available there:

a.       Zoom to selection: This will zoom in the histogram by expanding the range between the two histogram poles.

b.       Fit to graph: This will zoom out the histogram and return it to the default state (the state when you opened the histogram dialog).

c.       Fill Range view (for 16 and 12 bit images only): This will zoom in the histogram by displaying 256 color at the histogram window, and it enable a horizontal scroll bar at the bottom of the histogram window that allow the user to scroll the histogram values in order to see the rest of the histogram. Since the histogram is expanded, the two poles will be sometimes out of reach. That is why by left clicking on the mouse anywhere in the histogram window, you can drag the left pole, while by clicking the left mouse button while holding the SHIFT key, you can drag the right pole.

 

(2)     Selection Information, which represents the information about the histogram window pole, the pole current color, and the number of pixels with this color. The user also can update the pole manually through the edit box.

 

(3)     Apply to option, which allow the user apply the histogram updates on the actual bitmap data or on the bitmap LUT (only with 16 and 12 bit images), the user also can check or uncheck whether he wants to see the effect directly on the image, by checking or unchecking the Show preview check box.

 

(4)     Statistical information on the histogram Selection. Which contains all the information contained in the range between the two poles in the histogram window:

a.       The pixel count.

b.       The percentage of the pixels in the range to the whole image.

c.       The range of the selection.

d.       The mean.

e.       The median

f.         The standard deviation.

 

(5)     The Channel, which determine the channel that will be modified or altered. Keep in mind that when the image is grayscale, this option will be dimmed since you only have one channel (the gray channel).

 

(6)     The Selection color group, which allow the user to change the color of the inner range and the outer range of the histogram in the histogram window (see point 1). The inner range is the range that lay between the two poles in the histogram window, while the outer range represents the area that lay outside those poles.

 

(7)     Selection Type: which represents the method of how to determine the histogram selection:

 

a.       All: the whole histogram is the selection.

b.       Inner Area: the area between the two histogram window poles in the selection.

c.       Left area: the area to left of the left pole.

d.       Right area: the area to right of the right pole.

 

The Hide Options button minimizes the size of the dialog in order to allow the user to have a better look at the affected image.

 

(8)     Segmentation information (part of segmentation tab): Which does some further manipulation on the selected area of the histogram either by:

 

a.       Setting all the values outside the selection to Start or end color based on where the outside value lay (this is the case when the user checks the gradient check box)

b.       or by segmenting (Thresholding) the image by setting all the values inside the selection range to the inner color, (which can be specified by clicking on the start color button), and setting all the values outside the selection range to the outer color (which can also be specified by clicking on the end color button). (This is the case when the user checks the threshold check box).

 

(9)     Mouse statistical information, which represents the information of the current color that has the mouse hovering over on the histogram window. The information are:

 

a.       Count: This represents the number of pixel with this color.

b.       Percent: the percentage of the pixel value in term of how close it is to the maximum value.

c.       Level: the actual color level.

 

 

 

Notice that you have 4 tabs that you can click on to get some further tools for histogram manipulation and statistical information:

 

a.       Segmentation tab: this was covered above.

b.      Gray Distribution tab.

c.       Noise filter tab.

d.      And rescale tab.

 

Keep in mind that each tab can have its own poles values. So for the example the poles positions can be 100 and 200 when selecting the segmentation tab and 90, 230 for the noise filter tab.

 

(10) Gray distribution Information: this feature remaps the colors of the image according to a specific algorithm, range, color palette, and some extra modifiers. This feature consist of:

a.       Function Type: which represents the type of the algorithm used to remap the color of the image based on two poles of the histogram window (see histogram window (1)). The algorithms are:

                                                               i.      Linear: Normal distribution.

                                                             ii.      Exponential: Exponential distribution. The user is allowed to set a factor to determine the shape of the exponential function.

                                                            iii.      Log: logarithmic distribution. The user is allowed to set a factor to determine the shape of the logarithmic function.

                                                            iv.      Sigmoid: Sigmoid distribution. The user is allowed to set a factor to determine the shape of the Sigmoid function.

b.       The Start and the end color: which are used to determine the start and the end color that will be used when remapping the image colors. If a colored image is choosen, then the red start color will be the red component of the start color button. And the red end color will be the red component of the end color button..etc.

c.       Selection only check box: which allow the user to determine whether to remap the selection only and set all the outside value to zero or not.

d.       User predefined Palette: allow the user choose whether to use a predefined palette (by clicking on Custom palette check box) when remapping the image colors.

 

 

 

(11) Filter (Noise) information: this feature handles the values outside the selection. (The selection can be controlled through the two poles of the histogram window - see above  and the Selection Type combo box). There are 4 types of handling:

a.       Start and End Points: Replace the value outside the range with the values of the two poles of the histogram window. All the colors to left of the left pole will have the color of the left pole, and all the colors to the right of the right pole will have the color of the right pole.

b.       Min and Max intensities: Replace the value outside the range with the minimum and maximum intensity values. All the colors to left of the left pole will have a minimum intensity color, while all the colors to the right of the right pole will have the maximum intensity color.

c.       Replacement color: Replace the values outside the range with the replacement color, which can be changed through the selection color button.

d.       Zero: Replace the values outside the selection range with black.

 

 

(12) Rescaling Information: This will alter the values that lies in the selection range, by either shifting those values by a specific amount, or by remapping them to another range:

a.       Shift: This will Add the value "Shift amount" to the selection range specified by the two poles of the histogram window (refer to histogram window (1)). You can either add or subtract:

                                                               i.      By Checking the "Shift right" radio button, you will add the value "Shift amount" to the colors in the selection ranges.

                                                             ii.      By checking the "Shift left" radio button, you will subtract the value "shift amount" from the colors in the selection range.

b.       New Start/End: This will add two extra poles to the histogram window, the values inside the range of those extra poles will replace the values inside the selection range.
