LEADTOOLS HTML5 Demos

    The LEADTOOLS HTML5 / JavaScript toolkit provides developers with the power of the LEADTOOLS imaging SDKs for, cross-platform development.

    The LEADTOOLS HTML5 / JavaScript online demos show a portion of LEADTOOLS. Features include extended file format support (TIFF, GIF, JPEG2000, JPEG, and more), automated annotations, image processing, barcode recognition, OCR and DICOM image display.

    Download the LEADTOOLS Imaging SDK Evaluation that Includes the HTML5 / JavaScript toolkit.

    Document Viewer Demo

    Demonstrates the LEADTOOLS HTML5 Document Viewer functionality.

    • Load document formats as Image or Svg
    • Annotations support
    • Interactive Modes including Magnify Glass, Pan, Zoom and more
    • Multi-touch and gesture support for mobile devices and tablets

    SVG Demo

    Demonstrates the LEADTOOLS Svg functionality.

    Load document image file formats as SVG.

    Barcode Demo

    Demonstrates the LEADTOOLS Barcode functionality.

    Detect 1D and 2D barcodes including UPC/EAN, DataBar, Code 128, Data Matrix, PDF417, QR Code and more.

    OCR Demo

    Demonstrates the LEADTOOLS OCR functionality.

    Convert raster images to document files.

    Scanning Demo

    Demonstrates the LEADTOOLS Web Scanning functionality.

    • Scan and upload images from TWAIN devices in HTML, browser-based applications.
    • Apply image processing functions on scanned images.
    • Print scanned images.
    • NOTE: Requires downloading local scan application for Windows

    Medical Web Viewer Framework Demo

    A zero footprint solution for viewing DICOM Images with Window Level, Stack, annotations/markup and multi touch support for Mobile, Tablet & Desktop.

    Annotations Demo

    Demonstrates the LEADTOOLS HTML5 Annotations (image markup) functionality.

    Annotation objects: Rectangle, Ellipse, Hilite, Line, Redact, Text, Ruler, Protractor, CrossProduct, Image, TextPointer, Point, Curve, Point, Polygon, Polyline

    Viewer Demo

    Demonstrates the LEADTOOLS HTML5 / JavaScript viewer functionality.

    • Full Featured, zero footprint Image Viewers for any browser, platform and device supporting HTML5
    • Interactive Modes including Magnify Glass, Pan, Zoom and more
    • Multi-touch and gesture support for mobile devices and tablets
    • Basic Image Processing

    Thumbnail and Bookmark Demo

    Demonstrates the LEADTOOLS HTML5 multi-page image and PDF Bookmarks support

    • View and browse multipage images such as PDF and TIF
    • View and use PDF bookmarks and links

    Image Processing

    Demonstrates the LEADTOOLS Image Processing functionality.

    Apply Image Processing filters to images

    Bank Check Reader

    Demonstrates using the LEADTOOLS Forms Recognition and Processing SDK to recognize bank checks.

    Detect the location of and recognize the data in fields from bank check images, including:

    • MICR data
    • Payee information
    • Check date
    • Amount
    • Signature area
    • Payee information

    MRTD Reader

    Demonstrates using the LEADTOOLS Forms Recognition and Processing SDK to recognize MRTD zones in passports.

    Recognize and extract passport features and fields.
